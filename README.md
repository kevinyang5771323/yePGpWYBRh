# 数据分析可视化系统 python718

本项目是一个基于Python、Vue.js和MySQL开发的数据分析可视化系统，致力于提供清晰直观的数据展示，帮助用户深入理解数据背后的价值。

## 技术栈

- 后端：Python
- 前端：Vue.js
- 数据库：MySQL

## 功能模块

- **评论分析**：对用户评论进行深度分析，挖掘用户真实反馈。
- **总览**：全局视角查看数据概况，快速把握数据趋势。
- **词云图**：通过词云图展示数据中的关键词，直观了解数据热点。

## 系统角色

- 用户：系统的主要使用群体，可以通过系统提供的功能模块进行数据分析。

## 运行环境

- 后端：Python 3.8+
- 前端：现代浏览器（建议使用Chrome、Firefox等）
- 数据库：MySQL 5.7+

## 部署步骤

1. 准备Python运行环境，安装依赖库。
2. 配置MySQL数据库。
3. 部署前端Vue.js项目。
4. 启动后端服务。
5. 通过浏览器访问前端项目。
